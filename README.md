## Getting Started

### Prerequisites

- Node.js (v18.x or higher recommended)
- npm or yarn

### Installation

```bash
git clone https://github.com/ABHAY627/Next.js.git
cd Next.js
npm install
# or
yarn install
```

## Usage

### Development

```bash
npm run dev
# or
yarn dev
```
Open [http://localhost:3000](http://localhost:3000) to view the app in your browser.

### Production

```bash
npm run build
npm start
# or
yarn build
yarn start
```
